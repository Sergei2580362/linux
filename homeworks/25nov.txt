1. Что такое Linux и для чего он применяется.
Это оперционная система, используемая также для серверов из-за в первую очередь из-за своей безопасности (отсутсвия вирусов).

2. Чем отличается CLI от GUI.
GUI - это взаимодействие с ситемой через графический интерфейс
CLI - это взаимодействие с ситемой через коммандную строку (более практичный режим)

3. Что такое SSH.
Это протокол удаленного доступа и одноименная команда для подключения (secure shell)

4. Как сгенерировать пару RSA-ключей и зачем это делать.
Через командную строку: ssh-keygen -t rsa
Для беспарорльной авторизации в разчичных системах

5. Каково назначение команды df ?
Получение нформации об использовании дискового пространства файловой системы

6. Как посмотреть публичный ключ на своем компьютере? 
cat ~/.ssh/id_rsa.pub

7. Какие задачи решаются с помощью команды 'gzip'.
Задачи по сжатию (архивированию) файлов для высвобождения дополнительно свободного пространсва файловой системы
